== README

LilyCMS é um gerenciador de sites open-source, em vista de ajudar e contribuir com a comunidade Ruby on Rails, ele possui CRUD para conteúdo, banners, contatos, categorias. Além de possuir um formulario para recebimento de sugestões integrado com o MailCatcher instalado e configurado. 

- Devise https://github.com/plataformatec/devise
- MailCatcher https://github.com/sj26/mailcatcher
- Will_Paginate https://github.com/mislav/will_paginate
- rspec https://github.com/rspec/rspec
- PaperClip https://github.com/thoughtbot/paperclip#image-processor
